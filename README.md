# Repositório para curso da Alura: 
## Protocolos e arquitetura para construção de agentes: MCP, A2A, AG-UI e Backend for Agents (BFA)

Um agente bancário simples para abertura de conta e cartão de crédito utilizando:
- LLM local no LM Studio
- Docker
- Multiagentes
- Langchain
- Langgraph
- Langsmith
- Agent to Agent (A2A)
- Model Context Protocol (MCP)
- Frontend AG-UI
- Backend for Agents (BFA)
